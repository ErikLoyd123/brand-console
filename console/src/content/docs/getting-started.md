This console is the front end for a local content engine. It runs on your machine, talks to a local API and a local database, and gives you a place to see and steer the content pipeline end to end.

## What it does

The pipeline moves an idea through four stages:

1. **Discover.** A discovery agent reads your configured RSS feeds and other sources, scores each item against your voice card and pillars, and drops promising ones into the idea queue.
2. **Queue.** You review queued items in the Queue view. Some need your take (a personal angle only you can supply); the rest are ready to draft as is.
3. **Draft.** A drafter writes a post in your voice, from the queued idea. Every draft goes through a review gate before it is considered done.
4. **Publish.** You read the draft, decide if it's ready, and either copy it out to publish by hand or, once LinkedIn is connected, publish it directly from the console.

Alongside the short-form pipeline, the **Articles** view is a long-form lane: pieces for the web (how-tos, explainers, comparisons and more) built section by section in the same voice, reviewed by the same gate, and exported as Markdown. See **Long-form articles** in this Docs section.

Nothing in this pipeline auto-publishes. Every post reaches a human decision point before it goes anywhere public.

## Where to go next

- **The profile switcher** (top of the sidebar) shows which profile — a person or a brand — the whole console is working as, and is where you create a new one.
- **Voice** (Inputs) to see the voice card and pillars everything is drafted against — and, while a profile is incomplete, to run the guided setup.
- **Queue** to see what's waiting for a decision right now.
- **Connections** (System) to link LinkedIn if you want direct publishing instead of copy-paste. Optional — copy-paste always works.
- **Database** (System) if you want to look at the raw tables behind all of this.

If you're setting this up for the first time, start with **Setup > Profiles & brands** in this Docs section: create a profile from the sidebar switcher, then let the Voice page walk you through setup. Connecting LinkedIn can wait until you have something to publish.
